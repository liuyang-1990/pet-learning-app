# Use Spoken Examiner Flow for Daily Speaking Practice

Daily Speaking Practice will use a Spoken Examiner Flow instead of text-first prompt cards. In Speaking Part 1, the AI examiner automatically speaks each question and dynamic follow-up, the learner answers by recording first, Text Hints are hidden by default and recorded only as a non-punitive listening support signal, and feedback is given after the short exchange rather than after every answer.

Speaking Part 2 follows the same listening-first pattern for the task instruction while keeping the Part 2 Image visible; it does not use dynamic follow-up in the first version. This keeps daily practice low-pressure while training listening comprehension, spoken response habits, and PET-relevant feedback separately from a future Mock Test mode.
